# Projeto: Gerenciamento de Pouso em Marte (Nave Aurora)

## Introdução
Este projeto foi desenvolvido como parte das atividades acadêmicas focadas em lógica de programação e análise de dados. O sistema simula o controle de trajetória e estabilização da base **Aurora Siger** em sua aproximação ao planeta Marte, aplicando conceitos de governança e eficiência técnica.

## O que o projeto faz?
O código processa informações críticas para garantir um pouso seguro e a organização da colônia marciana, realizando:

* **Cálculo de Trajetória:** Processamento de dados de telemetria para ajustes de aproximação.
* **Gestão de Módulos:** Organização lógica dos componentes de pouso e estabilização da base.
* **Análise de Dados:** Aplicação de funções matemáticas para prever e tomar decisões baseadas nos dados coletados.
* **Aplicação de Lógica:** Uso de algoritmos de busca e ordenação para priorizar alertas e gerenciar recursos limitados.

#  Como Executar o Projeto

Siga os passos abaixo para acessar e rodar o código fonte em sua máquina.

# 1. Pré-requisitos
* Ter o **Python 3* instalado.
* O arquivo do código (`.py`) baixado em uma pasta de sua preferência.

# 2. Acessando o terminal
Abra o terminal do seu sistema (Prompt de Comando ou PowerShell no Windows, Terminal no macOS/Linux), ou utilize uma IDE de sua preferência, e navegue até a pasta onde o arquivo está salvo.

# 3. Executando o código Terminal
Para rodar o script, utilize o comando abaixo:
python nome_do_seu_arquivo.py
